### Setup
1. First set up `Raster2Seq`. At the time of writing, this is currently a private repository; reach out to Hao Phung for more information (htp26@cornell.edu).
2. Run the following script to clone the repo:
```bash
git clone https://github.com/chanjbc/floorplan-reconstruction.git
cd floorplan-reconstruction
```
3. Run the following script to create a conda environment:
```bash
conda env create -f environment.yml
```
